
# Eukaryote

## Install

```
git clone git@github.com:coreyferguson/eukaryote.git
cd eukaryote
npm install && bower install
```

## Usage

### Build and test in PhantomJS

```
grunt
```

### Continuous testing with Karma

```
grunt test:continuous
```
